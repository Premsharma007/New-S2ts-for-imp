When you upload an audio file and click "Run Full Pipeline" with all toggles enabled:

1. ASR Stage (if enabled):

Your OfflineTranscribe.exe will run minimized

Tamil speech → Tamil text conversion happens

Output: Proj-<filename>/<filename>-ASR.txt

2. Text Cleaning Stage (if enabled):

Opens Chrome with your selected AI engine (Gemini/ChatGPT)

Sends ASR text + correction prompt via GUI automation

Receives cleaned Tamil text back

Output: Proj-<filename>/<filename>-ASR-Clean.txt

3. Translation Stage (for each enabled language):

Hindi: Cleaned Tamil → Hindi translation

Kannada: Cleaned Tamil → Kannada translation

Telugu: Cleaned Tamil → Telugu translation

Each creates: Proj-<filename>/<filename>-<Language>-Translated.txt

4. TTS Stage (for each enabled language):

Takes translated text + your reference audio

Generates voice clone audio using IndicF5

Each creates: Proj-<filename>/<filename>-<Language>-TTS.wav



data/projects/Proj-my_audio/
├── my_audio-ASR.txt              # Raw Tamil transcription
├── my_audio-ASR-Clean.txt        # Corrected Tamil text
├── my_audio-Hindi-Translated.txt # Hindi translation
├── my_audio-Hindi-TTS.wav        # Hindi audio
├── my_audio-Kannada-Translated.txt
├── my_audio-Kannada-TTS.wav
├── my_audio-Telugu-Translated.txt
